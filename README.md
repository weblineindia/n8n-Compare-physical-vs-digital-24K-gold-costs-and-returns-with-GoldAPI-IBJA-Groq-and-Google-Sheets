# n8n-Compare-physical-vs-digital-24K-gold-costs-and-returns-with-GoldAPI-IBJA-Groq-and-Google-Sheets
Compare physical and digital gold investments with this n8n workflow automation. It fetches live gold prices, factors in GST and charges, analyzes market sentiment with Groq AI, calculates an investment efficiency score, and logs actionable insights to Google Sheets for smarter buying decisions.
